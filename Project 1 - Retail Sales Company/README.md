# Retail Sales Company Analysis

This project is a **Retail Sales Analysis Dashboard** built using an Excel dataset. It analyzes sales performance across stores, product categories, customers, and time periods to help understand business trends and revenue contribution.

## Project Overview

The dataset contains retail sales records with the following fields:
- Order ID
- Date
- Store
- Customer
- Category
- Product
- Quantity
- Unit Price

The analysis covers:
- Total revenue
- Sales by store
- Sales by category
- Monthly sales trend
- Top customers by revenue
- KPI summary

## Dataset Summary

- Total Rows: 100
- Stores: Ahmedabad, Rajkot, Surat, Vadodara
- Categories: Laptop, Mobile, Tablet
- Products: HP 15, Dell Inspiron, Samsung A35, Redmi Note 14, Lenovo Tab

## Key Insights

- Ahmedabad generated the highest revenue among all stores.
- Laptop sales contributed the largest share of total revenue.
- January showed strong sales performance compared to February.
- Rahul Mehta, Jay Patel, Amit Shah, Priya Patel, and Neha Joshi were among the top customers by revenue.

1. Download the Excel file.
2. Open it in Microsoft Excel.
3. Explore the dashboard sheets and pivot summaries.
4. Use the filters to analyze sales by store, category, customer, and date.

## Project Purpose

The purpose of this project is to practice retail sales analytics and create a simple business dashboard that highlights revenue trends and customer insights.
