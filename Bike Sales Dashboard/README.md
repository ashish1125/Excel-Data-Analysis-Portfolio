# Bike Sales Dashboard

## Project Overview

This project analyzes customer bike purchase behavior using Excel.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

## Process

1. Cleaned the raw dataset
2. Created a working sheet
3. Built pivot tables
4. Created charts
5. Designed an interactive dashboard
