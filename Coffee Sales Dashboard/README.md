# ☕ Coffee Sales Dashboard (Excel Project)

## 📌 Project Overview

This project is an interactive Coffee Sales Dashboard built entirely in Microsoft Excel.

The dashboard helps analyze coffee sales performance across different countries, customers, coffee roast types, package sizes, and loyalty card usage. It provides a clear visual representation of sales trends and customer purchasing behavior.

---

## 🎯 Objective

The goal of this project was to practice and improve Excel skills by building a professional business dashboard from raw sales data.

The project involved:

- Data Cleaning
- Data Transformation
- Data Analysis
- Dashboard Design
- Interactive Reporting

---

## 🛠 Tools & Features Used

### Excel Features

- Pivot Tables
- Pivot Charts
- Slicers
- Timeline Filter
- XLOOKUP
- Data Cleaning
- Data Formatting
- Conditional Formatting
- Dashboard Design

---

## 📊 Dashboard Insights

The dashboard allows users to:

- Track total sales over time
- Compare sales by coffee type
- Analyze country-wise sales performance
- Identify top customers
- Filter data using:
  - Roast Type
  - Package Size
  - Loyalty Card Status
  - Order Date

---

## 📈 Key Metrics Included

- Total Sales Trend
- Sales by Country
- Top 5 Customers
- Coffee Type Performance
- Customer Purchase Analysis
